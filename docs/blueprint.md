# **App Name**: Udaan Aerospace

## Core Features:

- Dynamic Universe Background: Implement a responsive, animated universe background with twinkling stars, glowing nebulae, and parallax scrolling to evoke a sense of flight, utilizing Next.js and Tailwind CSS.
- Interactive Project Showcase: Display aeromodelling projects using interactive cards or 3D previews dynamically fetched from Firestore, allowing users to explore project details and achievements.
- Event Management: Manage upcoming and past events, synced with Firestore to display a chronological list of activities, including date, location, and description.
- Secure User Registration and Authentication: Enable user registration and authentication using Firebase Authentication, securely storing user profiles and engagement data in Firestore.
- Core Team Display: Create dynamic profile cards for core team members, displaying names, roles, and photos fetched from Firestore, with options to include contact information.
- Animated Achievement Timeline: Showcase club awards, milestones, and notable projects in an elegantly animated timeline format, with data dynamically sourced from Firestore.

## Style Guidelines:

- Primary color: Neon cyan (#00FFFF) to represent innovation and the spirit of flight.
- Background color: Deep navy (#191970) with black gradients to create a cosmic feel, desaturated at 20%.
- Accent color: Light cyan (#E0FFFF), a hue analogous to neon cyan, to highlight interactive elements and create contrast.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text, to convey a futuristic and tech-oriented aesthetic.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist icons that reflect aerospace themes. Icons should feature subtle glow effects.
- Implement a responsive, mobile-first layout with smooth animations powered by Framer Motion or GSAP ScrollTrigger.
- Use subtle glowing hover effects and depth transitions to create a cinematic user experience. Ensure all animations are smooth and optimized for performance.
